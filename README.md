# minesweeper
Jocul clasic minesweeper, ce va avea ca functionalitati: generarea random a hartilor, optiunea de a marca cu stegulet sau alegerea nivelului de dificultate (Usor, Mediu, Greu) Precum si optiunea de a descoperi o arie mai mare a hartii.
  Jocul nu are grafica, deci se poate rula direct din compilator.
  Pe github sunt unele fisiere ce contin incercarile mele mai putin reusite ale funtiilor. Mentionez ca tema pe care doresc sa o prezint, este cpp-ul Minesweeper-Final;
  Download and run.
